# EKS 기반 분산 데이터베이스

:::info

**참고**: 분산 데이터베이스 및 쿼리 엔진 블루프린트는 현재 개발 중입니다.
배포 예제가 저장소에 추가되면 문서가 업데이트될 예정입니다.

:::

Kubernetes에서 분산 데이터베이스와 쿼리 엔진을 실행하면 대량의 데이터를 실시간으로 관리하고 처리하려는 조직에 다양한 이점을 제공할 수 있습니다. Kubernetes는 자동 스케일링, 롤링 업데이트, 자가 복구 기능과 같은 기능을 제공하여 이러한 시스템의 고가용성과 안정성을 보장합니다. **Apache Cassandra, Amazon DynamoDB, Apache Presto**를 포함한 여러 인기 있는 분산 데이터베이스와 쿼리 엔진이 이러한 사용 사례를 지원하기 위해 등장했습니다.

이러한 시스템은 대량의 데이터를 실시간으로 쉽게 관리하고 처리할 수 있게 해주며, 확장성, 고가용성, 실시간 데이터 처리와 같은 기능을 제공합니다. Kubernetes의 강력한 기능을 활용함으로써 조직은 이러한 복잡한 시스템의 배포, 스케일링 및 관리를 단순화하고 자동화하여 비즈니스의 다른 영역에 집중할 수 있는 리소스를 확보할 수 있습니다. 성장하는 도구 생태계와 광범위한 사용 사례 지원을 통해 Kubernetes는 프로덕션 환경에서 분산 데이터베이스와 쿼리 엔진을 실행하기 위한 점점 더 인기 있는 선택지가 되고 있습니다.

## 파트너 솔루션을 선택해야 하는 이유

### 엔터프라이즈 지원
미션 크리티컬 워크로드에 대해 AWS 파트너로부터 상업적 지원 및 SLA를 받으세요.

### 검증된 통합
파트너 솔루션은 Data on EKS 데이터 스택과 원활하게 작동하도록 설계되었습니다.

### 배포 가속화
사전 구축된 통합을 활용하여 복잡한 데이터 플랫폼의 프로덕션 적용 시간을 단축하세요.

### 확장된 기능
오픈소스 데이터 스택을 보완하는 전문 도구 및 플랫폼에 액세스하세요.

## 파트너 카테고리

파트너 솔루션은 모든 Data Stack 카테고리에 걸쳐 있습니다:

- **프로세싱(Processing)**: 고급 Spark 최적화, GPU 가속, 특수 컴퓨팅
- **스트리밍(Streaming)**: 엔터프라이즈 Kafka 배포판, 스트림 처리 플랫폼
- **오케스트레이션(Orchestration)**: 관리형 Airflow 서비스, 워크플로우 자동화
- **데이터베이스(Databases)**: 상용 데이터베이스, 특수 데이터 저장소 (Altinity ClickHouse, Aerospike)
- **관측성(Observability)**: APM 도구, 비용 관리, 보안 플랫폼
- **AI/ML**: 벡터 데이터베이스, 피처 스토어, 모델 서빙
